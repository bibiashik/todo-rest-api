# Todo REST API

A Spring Boot REST API project for Todo Management using CRUD operations.

## Features
- Create Todo
- Get Todo by ID
- Get All Todos
- Update Todo
- Delete Todo

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Maven

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /todos | Create Todo |
| GET | /todos/{id} | Get Todo By ID |
| GET | /todos | Get All Todos |
| PUT | /todos | Update Todo |
| DELETE | /todos/{id} | Delete Todo |
